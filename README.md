# LMS FRONT

## Install Yarn 

```npm -g install yarn```

## Install dependencies

```yarn install```

## Copy the .env.example file to .env and fill the variables

```cp .env.example .env```

## You're all set ! Run the project

```yarn dev```

## Run the tests

```yarn test```

## Run the tests with coverage

```yarn test:coverage```

## Access coverage report

```open coverage/lcov-report/index.html```
